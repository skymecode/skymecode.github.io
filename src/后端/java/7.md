# 接口和抽象类

## 抽象类概念

- abstract修饰类——抽象类
    - 包含抽象方法的类
- abstract修饰方法——抽象方法
- 没有方法体的方法

### 接口小结

---

1. 接口不能创建对象
2. 接口中所有方法都是抽象方法，方法是public abstract
3. 接口中的属性都是常量，默认是public static final
4. 一个类可以实现多个接口
5. 接口支持多态的一种形式，子类和接口之间支持方法的重写

jdk8后可以有的方法体

```java
default void method(){

}
```

也可以有static方法,也可以有方法体。
