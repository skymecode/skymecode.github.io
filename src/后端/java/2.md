# 面向对象基础

### 面向对象和面向过程

- 面向过程:面向过程可以理解为流水线上制造一个产品，需要每一步按顺序进行，并且需要知道每一步的细节。
- 面向对象:面向对象可以理解为黑盒,就和我们大部分人使用一个电子产品，我们不需要知道它内部如何构造的,只需要知道如何使用。

### 类和对象

- 类:是一种描述,可以是种类,像人，动物等等。
- 对象:有着具体的属性，方法，比如这个人叫小明，他能打篮球。

### 类和对象的定义

- 定义类:

```Java
class Dog {
    String name;
    String color;
    int age;
}//这里前面加上private最好，因为一般不会直接访问成员变量
```

- 定义对象:

```Java
public class test1 {
    public static void main(String[] args) {
        Dog dog =new Dog();
    }

}
```

### 成员变量和局部变量

- 成员变量:用于定义类
- 局部变量:一般用在方法当中

### 构造函数

用于初始化对象,并且能够初始化对象中的属性,一般有无参构造和有参构造,但如果有参构造一般都要加上无参构造。

### this关键字

this关键字的this是指向的当前调用的对象

```Java
this.属性
this.方法
```

### 对象的堆栈模型

- 基础类型数据:int,char,boolean,long,byte
- 引用类型数据:数组，对象
