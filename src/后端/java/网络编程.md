## 1. 网络编程相关概念

### 1.1 什么是网络编程

---

- 网络编程可以让程序与网络上的其他设备中的程序进行数据交互
- 网络通信的基本模式:两台设备在网络上的互联，通过程序实现
    - C/S:两端都需要写程序,客户端程序与服务器端程序
    - B/S:只需要服务器需要写程序,客户端安装浏览器,网站

## 2.UDP通信

---

- 无连接的,不可靠传输的协议
- 封装数据包(数据源IP,目的地IP,端口),直接发送
- 数据包的大小限制64KB以内

```text
DatagramSocket类作为UDP协议的socket
socket:套接字(IP+端口)
```

DatagramPacket(byte[] buf,int len,InetAddress add,int port) 数据包对象,创建len的对象到指定主机的指定端口

DatagramSocket(),DatagramSocket(int port)数据包套接字,用于联通两个主机的应用的

```java
package 网络编程;

import java.io.IOException;
import java.net.*;

public class Sender {
    public static void main(String[] args) throws IOException {
        String a="hello world!";
        DatagramPacket datagramPacket = new DatagramPacket(a.getBytes(), a.getBytes().length, InetAddress.getByName("127.0.0.1"), 5000);
        DatagramSocket datagramSocket = new DatagramSocket();
        datagramSocket.send(datagramPacket);
    }
}
```

```java
package 网络编程.TCP;

import java.io.*;
import java.net.Socket;

public class Client {
    public static void main(String[] args) throws IOException {
        Client client=new Client();
        client.run();;
    }
    public  void run() throws IOException {
        //创建Socket
        Socket socket=new Socket("127.0.0.1",5000);
        FileInputStream fileInputStream=new FileInputStream("F:\\jack\\关于通知的json.txt");//字节流
        BufferedInputStream bis=new BufferedInputStream(fileInputStream);//字节缓冲流
        OutputStream os=socket.getOutputStream();
        BufferedOutputStream bos=new BufferedOutputStream(os);
        System.out.println("运行");
        int data;
        while ((data=bis.read())!=-1){
            bos.write(data);
        }
        bos.flush();
        socket.shutdownOutput();
        BufferedReader br = new BufferedReader(new InputStreamReader(socket.getInputStream()));//InputStreamReader转换流
        System.out.println(br.readLine());

        // 释放资源
        socket.close();
        os.close();
        bis.close();

    }
}
```

服务端:

```java
package 网络编程.TCP;

import java.io.*;
import java.net.ServerSocket;
import java.net.Socket;

public class Server {
    public static void main(String[] args) throws IOException {
        Server server=new Server();
        server.go();
    }
    public void go() throws IOException {
        ServerSocket serverSocket=new ServerSocket(5000);//监听5000端口

        while (true){
            Socket ss = serverSocket.accept();//阻塞方式
            System.out.println("接收到对象");

            File file=new File("F:\\png\\hello.txt");//存放文件的地址
            file.createNewFile();
            FileOutputStream fileOutputStream = new FileOutputStream(file);
            BufferedOutputStream bos=new BufferedOutputStream(fileOutputStream);
            InputStream is = ss.getInputStream();

            int data;
            while ((data = is.read()) != -1){
                bos.write(data);
            }

            PrintWriter pw = new PrintWriter(new OutputStreamWriter(ss.getOutputStream()));
            pw.println("传输成功！");
            pw.flush();

            // 释放资源
            bos.close();
            ss.close();
            ss.close();

        }

    }
}
```
