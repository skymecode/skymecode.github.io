### 图片和超链接：

1. #### 图片标签 img:
  ---
    1. img 单标签 浏览器中展示一张图片
    2. img标签展示图片,三个属性:
          - src:必需属性用于指示显示图片的位置，可以使用绝对路径或者相对路径
          - alt:非必需属性当图片无法加载时，用于替代显示的文字，开发中为了界面友好，推荐加上
          - title:非必需属性所有标签都可以加上该属性，当鼠标悬浮在标签上时，弹出的提示标注文字

  #### 超链接标签 a
  ---
  ```html
  ...标签用于用户点击后，向指定的地址发起请求，然后浏览器会打开所请求的页面。该过程称为页面跳转。
  ```
  ![](https://skyme-1307417630.cos.ap-chengdu.myqcloud.com/skyme/image-20230703113628644.png)

- a标签的target属性:用于指示浏览器如何打开请求的资源:
    - _self :默认值,标识在当前页面打开页面
    - _blank:浏览器会打开一个新选项卡,打开新页面
    - parent:在父窗口中打开新页面,如果没有父窗口,那么和_self效果一样
    - top：在顶层窗口中打开新页面,如果自己就是顶层窗口,那么和_self效果一样
    - iframe的name值: 浏览器会在指定的iframe标签中打开新页面

```html

		在当前页面打开百度

		在新选项卡打开百度

		在父窗口中打开百度

		在顶层窗口打开百度

		在下面的iframe中打开百度

```

### HTML列表：

#### 1. 列表类型:

- 主要分为无序列表，有序列表，定义列表三类
  ```html
  ...：无序列表，Unordered List。
  ...：有序列表，Ordered List。
  ...：定义列表，Definition List。
  ```

#### 无序列表ul

1. 无序列表中的各个项之间没有顺序级别之分，是并列的
2. ul标签需要搭配列表项li标签一起使用:
  ```html

              第一个表
              第二个表
              第三个表

  ```

#### 有序列表ol

```html
    有序列表

            第一个表
            第二个表
            第三个表

            第一个表
            第二个表
            第三个表

            第一个表
            第二个表
            第三个表

```

- type：已经不被推荐使用的属性，推荐使用CSS样式来设置。用于指示索引展示的方式，可取值：1,a,A,i,I。
- start：HTML5标准中再次被启用的属性，指示索引的起始值。
